# MazeBank Application
A banking application built with Java and JavaFX.

### Prerequisites
- Java JDK 11+
- Maven 3.6+
- JavaFX SDK

### Installation
```bash
git clone https://github.com/erenulutas0/mazebank.git
cd mazebank
```
# Run Application
```bash
mvn clean javafx:run
```

## Build
To build the project:
```bash
mvn clean package
```

## Technology Stack
- Java
- JavaFX - UI framework
- Maven - Build tool
- FXML - UI layout
- CSS - Styling

## Project Modules
- Main application entry point
- UI components and controllers
- Resource management
- Style definitions
